# Digital Theatre

Digital Theatre is a responsive frontend web project focused on showcasing theatre and movie-related content with a clean and modern UI.

## Features

- Responsive layout
- Clean and modern design
- Easy navigation
- Beginner-friendly structure

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Boostrap  

## Project Structure

digital-theatre/
├── index.html
├── css/
├── js/
└── assets/

## Getting Started

1. Clone the repository  
2. Open the project folder  
3. Run `index.html` in your browser

## Purpose

This project was built to practice frontend development skills and create a portfolio-ready web project.
